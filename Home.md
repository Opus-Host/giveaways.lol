## About

giveaways.lol is a powerful Discord bot designed to manage and automate giveaways in your Discord servers. Built with Java, it provides a robust and reliable solution for running giveaways with ease.

## Features

- Easy-to-use giveaway management
- Customizable giveaway settings
- Automatic winner selection
- Real-time notifications
- Multi-server support
